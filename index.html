Current status & milestone

Current milestone

2D card lifecycle plus contacts is feature-complete: a user can sign up for a free trial, create a 2D card, edit it, set it active, share it (QR, message, email, WhatsApp/Telegram, native share sheet), view it full-screen, manage their team under Users, and build a saved-contacts address book by scanning a contact's QR code — all talking to the real production backend with unchanged request formats.

Next milestone

Announcements & notifications. The announcement compose/receive flow and push notifications are the last pieces of the app's core day-to-day usage loop still outstanding.

Risks & blockers

  • A handful of backend quirks (e.g. a blob-path mismatch in the card-layout-edit endpoint) are being faithfully reproduced rather than fixed, per plan — a dedicated backend clean-up pass is scheduled for after full UI parity, not before.
  • 360° cards and Bluetooth P2P sharing need native plugin integration (panoramic viewer, BLE GATT protocol) — larger, riskier scope than the screens shipped so far.
  • In-app purchases require live Apple/Google store product configuration before that flow can be built end-to-end, not just app-side code.
